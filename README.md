# AVO — IT Company Website 🖥️

A fully responsive, multi-page front-end website for a fictional IT company built with pure **HTML**, **CSS**, and **JavaScript** — no frameworks, no libraries, no back-end. Designed as a portfolio/practice project to demonstrate modern web design and layout techniques.

---

## ⚠️ Known Non-Working Features

> This is a **dummy/static front-end project**. The following features are UI-only and not functional:

- **Contact Form** — The form on `contact.html` does not submit data anywhere. No back-end or email service is connected.
- **Blog Posts** — Blog cards are placeholder content and do not link to real articles.
- **Newsletter Subscription** — Any email input/subscribe button is decorative only.
- **Search functionality** — Not implemented.
- **Portfolio filters** — Filter buttons on the Work page are visual only and do not filter items.
- **Social media links** — Linked to `#` placeholders.
- **Animations/Slider JS** — Some JavaScript interactions may be partial or placeholder depending on the page.

## 🚀 Getting Started

No build tools or installations needed.

1. **Clone the repository**
   ```bash
   git clone https://github.com/ammarwaheed8/avo-it-website.git
   ```

2. **Open in browser**
   ```bash
   cd avo-it-website
   open index.html
   ```
   Or simply drag `index.html` into your browser.

3. **Or use Live Server** (recommended for VS Code users)
   - Install the [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) extension
   - Right-click `index.html` → **Open with Live Server**
